# Spotify Wrapped Complete

A lightweight Streamlit app that recreates and extends Spotify Wrapped using raw Spotify listening history.

Built to see full year stats and look back at past years.

Users can upload their “Extended Streaming History” JSON files from Spotify to explore full-year, historical, and all-time listening patterns.

Tools: Python (pandas, Plotly, Streamlit)

Live app: https://spotifywrappedcomplete.streamlit.app/
